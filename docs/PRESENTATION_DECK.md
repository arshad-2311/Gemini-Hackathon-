## Design Specifications

### Color Palette
```
Primary:     #6366f1 (Indigo - matches Gemini)
Secondary:   #10b981 (Emerald - success/positive)
Accent:      #f59e0b (Amber - highlights)
Background:  #0f0f23 (Dark blue)
Text:        #ffffff (White)
Muted:       #94a3b8 (Gray)
```

### Typography
```
Headings:    Inter 700 (Bold)
Subheadings: Inter 600 (Semi-bold)
Body:        Inter 400 (Regular)
Minimum:     24pt (for readability)
```

### Visual Guidelines
- Use rounded corners (16px radius)
- Add subtle drop shadows
- Include gradient backgrounds
- Animate transitions between slides
- Keep text minimal (< 50 words per slide)
- Use icons from Phosphor or Heroicons
